Hot Tab Options

Key Order

Below, you can configure the order of keys used to associate with. This order is used for pinning tabs in the next free space (;;) and for cycling through tabs (n, p).

URL Patterns

URL patterns for pinning with P@. Named captures (e.g., :docid) are filled from the tab URL. Use /* to match the rest of the path, query, and hash. Patterns are matched in order; the first match wins. If no pattern matches, the full URL is used as the pin pattern.

Shortcut Actions

This options page lets you assign command combos to command slots. Before assigning command combos, you'll need to assign shortcuts to however many command slots you'd like to use in Chrome's shortcut menu. You can type in chrome://extensions/shortcuts in the address bar to get to that menu.
